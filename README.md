# AnomalyDetector

AnomalyDetector — локальное приложение для анализа сетевого трафика

## Description
AnomalyDetector — это десктопное приложение для локального анализа сетевого трафика с использованием алгоритмов машинного обучения. Программа автоматически классифицирует сетевые потоки как нормальные или аномальные.

Поддерживаются только файлы формата pcap
